Dashbaords

Application

This application's purpose, is to give you the user, a brief overview of the dashbaord created in the Analytics Designer. With this application, you can quickly analyze data, and use this data to inspect flaws.

Mobile specific

Make sure that you create a basic dashboard, maintaining minimal data, as this application is used to quickly get an overview of your data. If flaws are found, make sure you have a dashboard for desktop, which is more detailed, and which lets you analyze your data even more!
